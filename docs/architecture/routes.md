## Edge Middleware

`middleware.ts` performs only the apex → `www` redirect (`cheersai.uk` → `www.cheersai.uk`, HTTP 308). It excludes `_next/static`, `_next/image`, `favicon.ico`, `robots.txt`, `manifest.webmanifest`, `sitemap.xml`. **Auth is not enforced here** — it is enforced in the `(app)` layout via [[overview#Auth Model summary|getCurrentUser]].
