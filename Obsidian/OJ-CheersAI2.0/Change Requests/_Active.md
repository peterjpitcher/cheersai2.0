## How to Create a Change Request

Add a `#change-request` tag anywhere in a documentation file followed by your request:

```markdown
#change-request
Add X feature to the Y section.
#end-change-request
```

Or add a `## Change Requests` section with `- [ ]` items to any document. Claude will pick these up on the next invocation.
